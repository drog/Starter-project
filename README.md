# Starter project


Getting Started
---------------

```bash
$ git clone git@github.com:drog/Starter-project.git starter-project
$ cd starter-project

# Install NPM dependencies
$ npm install && bower install

#Development
$ gulp serve-dev

#Prodcution
$ gulp serve-build

```
